# Karan Rawat Portfolio

A modern personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Live Website

GitHub Pages can be enabled for this repository to publish the site directly from the `main` branch.

## Sections

- Hero section with profile card and animated intro
- About section with education and experience
- Skills section with animated progress bars
- Real GitHub projects section
- Contact section with LinkedIn, GitHub, and email

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Local Run

Open `index.html` directly in the browser, or run a local server:

```powershell
python -m http.server 3000
```

Then visit `http://127.0.0.1:3000`.
